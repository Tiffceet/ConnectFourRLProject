Install instruction:
1. Install the dependencies by running `python -m pip install -r requirements.txt`
Note: If this doesnt work, you may install the module 1 by 1 following requirements.txt

How to run:
1. Run `python index.py` for DQN vs ARS
2. Run `python dqn_minimax.py` for DQN vs MiniMax
2. Run `python ars_minimax.py` for ARS vs MiniMax
